# DriversCars

Необходимо указать ConnectionString в файле AppSettingJson

При старке приложение создается 3 таблицы Drivers, Cars, DriverCarMaps
- Таблица со списком водителей
- Таблица со списком авто
- Таблица для привязки авто к водителям (т.к связь many-to-many)

## Взаимодействие

- CRUD операции с авто
- CRD операции с водителями (update отсутвстует т.к все поля водителя являются частью alternate key и являются readonly для EF)
- Привязка авто осуществляется во взаимодействии с контроллером DriversWithCars
